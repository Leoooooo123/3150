Introduction to pThread

Example using multi-threading

Problem

You are given two files, the program should output a sorted list of unique integers that are common between two files.

Usage

To run the pthread program:

./asg3-pthread [inputfile1] [inputfile2] [outputfile] [ThreadNum]

Sample File

input1.txt

10
2 1 0 8 8 6 1 5 7 0

input2.txt

15
4 3 2 9 0 2 6 5 2 2 3 1 1 1 1

If we execute the following:

./asg3-openmp testcases/C0/input1.txt testcases/C0/input2.txt res/output.txt 4

Result

0
1
2
5
6
